# Requirement

* Mac OS
* Android Studio Electric Eel | 2022.1.1 Patch 1
* Firebase 11.23.1
* Flutter 3.7.6
* Dart 2.19.3
* DevTools 2.20.1
* pod 1.12.0


# Installation

```bash
brew install cocoapods
pod setup
```

# Usage

```bash
git clone https://github.com/takuowake/ivent-reception-app.git
flutter run -d chrome
```
